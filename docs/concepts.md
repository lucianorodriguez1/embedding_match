## IA (Inteligencia Artificial)

Campo amplio que busca crear máquinas capaces de realizar tareas inteligentes.

## Machine Learning - ML (El motor)
La IA que aprende patrones a partir de datos sin ser programada explícitamente para cada regla.

* ### Aprendizaje Supervisado
El modelo aprende con datos etiquetados (ejemplo y respuesta correcta).
**Ejemplo:** Enseñar a la IA: "Esta foto es un perro" (etiqueta).

* ### Aprendizaje No Supervisado
El modelo encuentra patrones en datos sin etiquetas.
**Ejemplo:** Agrupar clientes de un banco por hábitos de compra similares.

* ### Feature (Característica)
Una propiedad medible de los datos usada para el análisis.
**Ejemplo:** En una casa: metros cuadrados, número de habitaciones.

## Deep Learning - DL (La potencia)
Un subcampo del ML basado en redes neuronales artificiales, ideal para datos complejos (imágenes, texto, audio).

* ### Red Neuronal Artificial
Sistema inspirado en el cerebro humano con capas de nodos.
**Ejemplo:** Capas que procesan bordes, luego formas, luego objetos en una foto.

* ### Weights (Pesos)
Valores numéricos dentro de la red que determinan la importancia de una conexión.
**Ejemplo:** Ajustar qué tanto influye el color rojo en la detección de un semáforo.

* ### Backpropagation
El método por el cual la red ajusta sus pesos cuando se equivoca.
**Ejemplo:** El modelo falla al identificar un gato, y corrige sus conexiones internas.

* ### GPU (Graphics Processing Unit)
Hardware especializado que acelera el entrenamiento de redes profundas.
**Ejemplo:** Las tarjetas NVIDIA usadas para entrenar modelos como GPT.

## NLP y Generative AI (Aplicaciones Modernas)
El procesamiento de lenguaje y la creación de nuevo contenido.

* ### Embedding
Representación numérica de palabras o datos en un espacio vectorial.
**Ejemplo:** "Rey" - "Hombre" + "Mujer" $\approx$ "Reina" en términos numéricos.

* ### Transformer
Arquitectura de red neuronal que revolucionó el NLP entendiendo el contexto.
**Ejemplo:** La tecnología base detrás de ChatGPT.

* ### Prompt Engineering
El arte de escribir instrucciones efectivas para obtener resultados de una IA.
**Ejemplo:** "¿Podés resumir este texto en 3 puntos clave con tono formal?"

## IA Engineering (Producción y Escalabilidad)

* ### RAG (Retrieval-Augmented Generation)
Técnica para darle a una IA acceso a datos privados o externos antes de responder.
**Ejemplo:** Conectar un chatbot de la universidad a un PDF con los reglamentos de exámenes.

* ### Vector Database
Base de datos especializada en almacenar y buscar embeddings (vectores) rápidamente.
**Ejemplo:** Pinecone, Milvus o ChromaDB, esenciales para RAG.

* ### Fine-Tuning
Re-entrenar un modelo pre-entrenado con datos específicos para una tarea.
**Ejemplo:** Tomar Llama 3 y entrenarlo específicamente para redactar contratos legales argentinos.

* ### MLOps (Machine Learning Operations)
Conjunto de prácticas para automatizar y gestionar el ciclo de vida de los modelos en producción.
**Ejemplo:** Usar Docker y Kubernetes para que tu IA atienda a 10.000 usuarios a la vez.
